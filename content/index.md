# Cybersecurity Portfolio

Welcome to my cybersecurity portfolio. This site showcases a combination of hands-on technical execution and security capability design, covering red team activity, incident response planning, and cloud security architecture.

The work presented here is based on controlled lab environments and portfolio case studies. All organisations, systems and data are either fictional, sanitised, or part of approved training environments.

---

## Featured Projects

### [[/projects/incident-response-program/|Enterprise Incident Response Program Design]]

Design and development of a structured Incident Response Program and Incident Response Plan aligned with NIST SP 800-61 and ACSC guidance.

This project includes:

* Full Incident Response Plan (IRP)
* Communications Plan
* Project Charter and governance structure
* Red, Blue and Purple Team exercise design
* Metrics including MTTD and MTTR

Focus areas:

* Incident lifecycle design
* Security governance and coordination
* Stakeholder communication planning
* Continuous improvement and metrics-driven response

### [[/projects/cloud-security-plan/|AWS Cloud Security Architecture & Uplift Plan]]

Design of a secure, scalable AWS environment for a growing business, including identity, network security, monitoring and incident response integration.

This project includes:

* End-to-end AWS architecture design
* Identity and access control model
* Security controls (WAF, GuardDuty, Security Hub, etc.)
* Logging, monitoring and alerting strategy
* Cloud incident response integration

Focus areas:

* Cloud security architecture
* Least privilege and identity design
* Detection and response capability in cloud environments
* Secure deployment and operational resilience
